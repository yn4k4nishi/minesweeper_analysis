マインスイーパーの数字があるところを

2行3列目の「1」から時計回りに順に
$b_i(1\leq i \leq 16)$
とし、

下図の青色の数字が振ってあるところを番号に沿って $x_j(1\leq j\leq 24)$
- 地雷がある場合 : $x_j = 1$
- 地雷がない場合 : $x_j = 0$

とおくと

$$
x_1 + x_1 + x_1 + x_1 = b_1 = 1 \\
x_3 + x_4 + x_5 = b_2 = 1 \\
... \\
x_{17} + x_{18} + x_{19} + x_{20} + x_{21} = b_{12} = 3 \\
... \\
$$
これらを行列にまとめる

$$
A \cdot X = B
$$
あとは掃き出し法なりで$x_j$を計算する
